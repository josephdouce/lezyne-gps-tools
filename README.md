# Features

- [x] Download Tiles
- [x] TCX to .LZR (Lezyne Route GPS File)
- [x] Automatically adds climb markers with climb info to your route
- [x] Climb Start xx km xx m Elevation
- [x] Climb xx km to go
- [x] Climb Finish
